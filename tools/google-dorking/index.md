# Google Dorking

## Overview
Google Dorking (Google Hacking) uses advanced search operators to find sensitive information exposed on the internet.

## Common Dorks
- site:
- intitle:
- inurl:
- filetype:

## Example Searches
```text
site:example.com filetype:pdf -- site:amazon.com filetype:pdf
intitle:"index of" "backup"
```
